# Claude Forge

A personal collection of handy plugins for Claude Code.

## Installation

```bash
# Add the marketplace
/plugin marketplace add illia1f/claude-forge

# Install a plugin
/plugin install <plugin-name>@claude-forge
```

## Plugins

| Plugin                         | Description                                                                                                                                                      |
| ------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [ideation](./plugins/ideation) | Multi-agent codebase ideation framework. Analyzes projects for improvements across security, UI/UX, performance, code quality, documentation, and code patterns. |
